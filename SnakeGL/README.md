SnakeGL
=======

Snake 3D game built using OpenGL and C++

## Requirements

- OpenGL (GL, GLU, GLUT)
- SOIL

## Controls

- Space - Change Camera
- P     - Pause
- S     - Start
- R     - Reset
- Q     - Menu
- Esc   - Quit
